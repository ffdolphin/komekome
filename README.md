# Rice Tracker Premium (お米価格追跡ツール)

Amazonとヨドバシカメラから「お米 5kg」の価格（送料込）をリアルタイムで検索し、指定した予算以下の商品をリストアップするツールです。

## 🌸 はじめて使う方へ (準備)

このツールを動かすには、**Node.js** というプログラムが必要です。

1. **Node.js のインストール**:
   - [Node.js 公式サイト](https://nodejs.org/ja/) へ行き、「LTS (推奨版)」をダウンロードしてインストールしてください。
   - インストールはすべて「次へ (Next)」で進めて大丈夫です。

## 🚀 使い方

### Windows の場合
1. このフォルダにある `launch.bat` というファイルをダブルクリックします。
2. 黒い画面（コマンドプロンプト）が立ち上がり、自動的に必要な準備が始まります。
3. 準備が終わると、ブラウザが自動的に開き、ツールが起動します。

### Mac の場合
1. `launch.command` をダブルクリックします。
2. ターミナルが開き、サーバーが起動します。

---

## 💡 ツールの操作方法

1. **価格上限を入力**: 画面の「価格上限」ボックスに、送料込みの予算（例: 3500円）を入力します。
2. **ボタンを押す**: 「価格をチェックする」ボタンを押すと検索が始まります。
3. **待つ**: 10〜20秒ほどで、Amazonとヨドバシの両方から条件に合うお米が表示されます。
4. **購入**: 気に入った商品があれば「商品ページを見る」を押して、各サイトで購入できます。

### ⚠️ 注意事項
- **ブラウザのウィンドウ**: 検索中、内部でブラウザが動いていますが、自動で閉じるのでそのままお待ちください。
- **検索の失敗**: インターネット接続が不安定な場合や、サイト側で一時的な制限がかかっている場合は、少し時間をおいてから再度お試しください。

## 🛠️ トラブルシューティング
- **「Node.js が見つかりません」と出る**: [公式サイト](https://nodejs.org/) からインストールをやり直してみてください。
- **画面が開かない**: `launch.bat` をもう一度実行してみてください。それでもダメな場合は、ブラウザで `http://localhost:3333` を直接開いてみてください。
